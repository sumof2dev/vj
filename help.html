How the RaveBox Raves

The engine uses a sophisticated multi-stage pipeline to turn raw sound into coordinated light shows.

⚡ The Audio Pipeline

The engine captures system audio using PulseAudio and performs real-time analysis:

  • 6 Frequency Bins: From Sub + Bass to Air / High, natively mapped for optimized processing and long-term average dominance.
  • Spectral Flux Tracking: Detects physical "impact" rather than just volume.
  • Normalization: A rolling 5-second window ensures the show is "full" regardless of track volume.

🧠 EDM Intelligence

RaveBox understands track structure by tracking musical "Transients":

  • Building: Detected when cumulative energy is rising fast.
  • Tension: The "Pre-Drop" state where volume/energy is high but bass vanishes.
  • Dropping: The massive impact state that triggers high-intensity patterns.

🎨 Layered DMX Logic

Commands pass through a 4-layer priority stack:

  1. Spatial & Frequency Logic: LFOs or Direct Frequency mapping (pinpointing the most dominant EQ bin) mapped to your stage dimensions.
  2. Base Presets: Animated scenes like "Lissajous" or "Neon Tunnel".
  3. System Triggers: Styles like "Machine Gun" or "Tearout" that override the base, and can even punch through global blackouts.
  4. Manual: Your overrides from the Setup UI.

Initial Setup

Advanced configuration for Raspberry Pi 5 hardware and system services.

1. Hardware Config

Edit /boot/firmware/config.txt to enable UART for RS485 and HifiBerry audio:

dtparam=uart0=on
dtoverlay=disable-bt
enable_uart=1
dtoverlay=uart0-pi5
dtoverlay=hifiberry-dacplus

2. System Services

The engine runs as three independent systemd units:

  • vj-server: Main DMX logic and WebSocket relay.
  • vj-launcher: The Flask API that starts/stops the engine.
  • vj-engine: The raw python processing core.

Run ./setup_service.sh to re-install or update these services.

3. SSL & Mobile Standalone

To use the Remote view on mobile without browser chrome, you must enable SSL. Run ./generate_cert.sh then accept the risk once in your mobile browser. This unlocks the "Install App" PWA features.
