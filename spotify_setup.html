Spotify API Setup

Connect your own Spotify Developer account to enable rich metadata and track analysis. This will bypass the built-in credentials.

1. API Credentials

2. Authorize Account

Click authorize to open Spotify login, then paste the URL you are redirected to below.

Setup Guide:
  1. Go to the Spotify Dev Dashboard.
  2. Create an App. In Settings, find ID and Secret.
  3. Add http://127.0.0.1:8888/callback to Redirect URIs.
  4. Enter credentials in Step 1 and Save.
  5. Click Open Spotify Auth in Step 2.
  6. Login, then copy the URL of the error/blank page you land on.
  7. Paste it in Callback URL and click Login.
